Install HDecode
1. http://htk.eng.cam.ac.uk/ftp/software/hdecode/HDecode-3.4.1.tar.gz
2. Extract HDecode to the same HTK folder (which was downloaded before)
3. $ make hdecode
4. $ make install 
5. $ make install-hdecode
